## 访问网站

[aigc.latentcat.com](https://aigc.latentcat.com)


## 本地运行、贡献文档

安装依赖

```shell
npm install
# or
yarn
```

将 `.env.example` 拷贝至 `.env.local` 并编辑：

```dotenv
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

运行 dev server：

```shell
npm run dev
# or
yarn dev
```

打开 [http://localhost:3000](http://localhost:3000) 实时浏览网页。
