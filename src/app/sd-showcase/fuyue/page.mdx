# 浮世绘 - LoRA



作者：<Authors names={["ciaochaos", "shichen"]} />

## 简介

此 LoRA 模型采用葛饰北斋《富岳三十六景（追加十景）》的 46 幅浮世绘训练而成，能较好还原浮世绘常见意象、画风和色彩搭配，且对有人物主体的图像生成有一定泛化能力。
